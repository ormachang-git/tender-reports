# 📋 每日政府採購標案配發系統

自動蒐集政府採購網（web.pcc.gov.tw）及台灣買賣網（taiwanbuying.com.tw）標案，
經 AI 媒合評分後每日自動發布至此，供業務團隊快速掌握投標機會。

## 篩選條件
- 📍 **地區**：北部（台北、新北、桃園、基隆、新竹）
- 🏗️ **類別**：工程類為主
- ❌ **排除**：最有利標
- 💰 **預算**：8,000 萬以下
- ⏰ **優先**：7 天內截止
